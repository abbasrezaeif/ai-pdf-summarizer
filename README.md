# AI PDF Summarizer

A Python application that extracts text from PDF files, generates a basic summary, and provides useful PDF statistics.

---

## 🚀 Overview

This project is a lightweight PDF processing tool built with Python.

It can:

- Extract text from PDF documents
- Generate a basic summary
- Save extracted text into a file
- Display useful PDF statistics
- Estimate reading time

The project is designed as a portfolio application and will continue to evolve with AI-powered features.

---

## ✨ Features

- 📄 Extract text from PDF files
- 📝 Generate a simple summary
- 💾 Save extracted text to a TXT file
- 📊 Display PDF statistics
- ⏱ Estimate reading time

---

## 🛠 Technologies

- Python 3
- PyPDF
- File Handling

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/abbasrezaeif/ai-pdf-summarizer.git
```

Go to the project directory:

```bash
cd ai-pdf-summarizer
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate the virtual environment:

macOS / Linux

```bash
source .venv/bin/activate
```

Windows

```bash
.venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶ Usage

Run the application:

```bash
python main.py
```
